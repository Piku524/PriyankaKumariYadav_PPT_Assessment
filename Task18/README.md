Task 18: Azure Virtual Machine 
Objective: Launch and configure a basic Azure Virtual Machine.  
Steps:
1. Login to Azure Portal.
2. Go to 'Resource Groups' → Create Resource Group ppt-rg.
3. Go to 'Virtual Machines' → Create VM → Name: ppt-vm.
4. Select Resource Group: ppt-rg.
5. Choose Image: Ubuntu Server 22.04 LTS or Windows Server 2022.
6. Select Size: Standard B1s (or default).
7. Set Username and Password (or SSH key).
8. Review + Create → Create.
9. Access the VM from CMD or SSH/RDP
