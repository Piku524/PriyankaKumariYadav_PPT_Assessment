Task 21: Microsoft Entra ID (Users & Roles)
Objective: Explore users and role management in Microsoft Entra ID.
Steps:

* Login to Azure Portal → Search 'Microsoft Entra ID'.
* Open Entra ID Dashboard → Click 'Users' → New User.
* Enter Name, Username, and basic details → Click Create.
* Go to 'Roles and Administrators' → Assign role: Reader to the user.
* Verify user and role assignment.
  Conclusion: User and role management in Entra ID has been successfully explored
