Task 19: Azure Storage Account (Blob, File, Table)
Objective: Create a storage account and explore Blob, File, and Table storage.  
Steps:
1. Login to Azure Portal.
2. Search 'Storage Accounts' → Create Storage Account.
3. Select Resource Group: ppt-rg.
4. Enter Storage Account Name: pptstorage123 (must be unique).
5. Select Region → Keep default settings → Review + Create → Create.
6. Access Storage Account → Click 'Containers' → Create new container → Upload a sample file.
7. Explore File Shares → Upload sample files.
8. Explore Tables → Create sample table and data.
