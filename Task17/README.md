Task 17: DynamoDB Table Creation (Basic)

Create a NoSQL table using DynamoDB to store and query items.

Objective:
Learn serverless database basics by creating a table, adding items, and querying them.

Steps:
Go to AWS Console → DynamoDB → Tables → Create Table.

Table Name: DynamoDBDemo

Partition Key: UID (String)

Sort Key: OID (String)

Leave other settings default → Click Create Table.

Once table is created, go to Explore Table → Items → Create Item.

Add sample items with attributes in String format.

Save items.

Use Search / Query to test retrieving items by Partition Key and Sort Key
