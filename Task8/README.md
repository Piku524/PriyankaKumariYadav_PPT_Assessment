Task 8: IAM Users and Groups

Objective: Understand access control and security policies by creating IAM users, groups, and attaching permissions.

Steps Performed:
1. Logged in to AWS Management Console and navigated to IAM.
2. Created IAM group named Developers and attached AdministratorAccess policy.
3. Created IAM users and added them to the Developers group.
4. Verified permissions by logging in with newly created users.
5. Attached additional policies to users.
