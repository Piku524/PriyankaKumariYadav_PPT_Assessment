Task 22: Azure Monitoring & Diagnostics
Objective: Monitor Azure resources using Azure Monitor and Metrics.
Steps:

* Open Azure Monitor → Metrics.
* Select a resource (VM or Storage Account).
* View CPU, Disk, and Network usage.
* Check 'Activity Log' for recent resource activities.
* Explore 'Alerts' section (basic overview).
  Conclusion: Azure resources can now be monitored and analyzed using Azure Monitor
