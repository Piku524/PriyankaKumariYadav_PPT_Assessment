Task 20: Azure Virtual Network
Objective: Create Virtual Network, Subnet, and configure basic networking.
Steps:

* Search 'Virtual Networks' → Click Create Virtual Network.
* Select Resource Group: ppt-rg.
* Enter VNet Name: ppt-vnet.
* Address Space: 10.0.0.0/16.
* Create Subnet Name: subnet1 → Subnet Address Range: 10.0.1.0/24.
* Review + Create → Create.
* Verify VNet → Ensure subnet1 exists with correct IP range.
  Conclusion: Basic Azure networking is configured with a Virtual Network and subnet
