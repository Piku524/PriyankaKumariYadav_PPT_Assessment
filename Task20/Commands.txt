Task 20: Azure Virtual Network (Basic Networking)
Objective: Create VNet and Subnet

Step 1: Search 'Virtual Networks' → Click Create Virtual Network
Step 2: Resource Group: ppt-rg
Step 3: VNet Name: ppt-vnet
Step 4: Address Space: 10.0.0.0/16
Step 5: Create Subnet:
   - Subnet Name: subnet1
   - Subnet Address Range: 10.0.1.0/24
Step 6: Review + Create → Create

Step 7: Verify configuration
Click VNet → Check subnets → Ensure subnet1 exists with correct IP range
