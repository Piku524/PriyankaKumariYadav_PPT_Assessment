Project: Static Website Hosting on EC2 using Nginx

Objective:
To host a static website on an Ubuntu EC2 instance using Nginx.

Steps Performed:
1. Created an Ubuntu EC2 instance.
2. Connected to EC2 using SSH.
3. Installed Nginx web server.
4. Downloaded website template using wget.
5. Unzipped the template files.
6. Copied files to /var/www/html directory.
7. Accessed the website using EC2 public IP.

Conclusion:
Successfully hosted a static website on EC2 using Nginx.
