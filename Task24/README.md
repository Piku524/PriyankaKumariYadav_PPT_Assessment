Task 24: SLA, Service Lifecycle & Subscription Management (Easy)
Objective: Understand Azure subscription, SLA, and service lifecycle.  
Steps:
1. Open Subscriptions → Active Subscription.
2. Review Subscription Overview (billing, quotas, region).
3. Go to 'Help + Support' → Search SLA.  
   - If SLA documentation is not visible in portal, open official link: https://learn.microsoft.com/en-us/azure/availability-sla/
4. Read SLA for Virtual Machines and Storage.
5. Open 'Service Health' → Check current resource health.
